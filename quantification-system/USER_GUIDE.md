# 量化决策系统 (Quantification System) 前端用户使用说明文档

欢迎使用 **量化决策系统 (v2.0.0)**。本系统是一个集 AI 驱动选股、技术量化分析及实盘信号验证于一体的高级金融工具平台。通过该手册，您将快速掌握系统各功能模块的使用方法。

## 1. 系统总览 (Dashboard)
**位置：** 导航栏 -> 仪表盘

仪表盘是系统的“神经中枢”，为您提供全方位的运行状态监控：
- **核心指标：** 查看当前持有的标的总数、AI 选股池的股票数量，以及活跃监控的实盘仓位。
- **系统健康度：** 显示数据同步的最新日期，并自动识别是否存在**数据缺失**的标的。
- **快捷指令：** 一键跳转至选股、分析、监控或数据任务。

## 2. 量化指挥中心 (Stock Selector)
**位置：** 导航栏 -> 选股与信号

该模块是系统逻辑的核心，利用 AI 模型对全市场进行扫描。

### 2.1 执行选股任务
1. **模型选择：** 在下拉框中选择预训练模型。
   - **XGBoost：** 侧重于特征相关性的非线性捕捉。
   - **LightGBM：** 高效的梯度提升框架。
   - **混合模型：** 同时运行多种模型进行综合评分。
2. **数量设置：** 设置扫描出的前 N 名优选股票（默认为 20）。
3. **执行扫描：** 点击“执行扫描”，系统将实时展示扫描进度。

### 2.2 查看结果
- **模型分类：** 结果按模型所属类型自动分页（如 XGB/LGBM）。
- **共振标签：** 标注为“共振”的股票代表其在多个维度或形态上同时触发了看多信号。
- **胜率指标：** 展示模型对该标的的预测胜率（红色代表信心度较高）。
- **因子快照 (PieChart 图标)：** 点击可查看计算该预测值所依赖的底层特征（如动量、波动率等）。
- **形态识别 (Lightning 图标)：** 查看系统对该标的进行的 K 线形态得分。

## 3. 技术分析引擎 (Analysis Engine)
**位置：** 导航栏 -> 技术分析

提供高度交互式的可视化图表分析工具。

### 3.1 核心功能
- **交互式 K 线：** 输入股票代码并按回车即可加载。
- **动态趋势线：** 
  - 支持**长期趋势**与**短期趋势**。
  - 自动绘制支撑位（红色线）与阻力位（青色线）。
- **形态标注：** 在 K 线图中直接标注“启明星”、“黄昏星”、“吞没形态”等经典 K 线组合信号。
- **市场微观结构：** 实时计算当前市场的趋势阶段（如 UP_TREND）、动量强度及近期结构是否发生反转。

## 4. 实盘验证中心 (Paper Trading)
**位置：** 导航栏 -> 实盘验证

旨在帮助用户记录并验证 AI 信号在真实市场中的表现。

### 4.1 持仓管理
- **部署监控：** 手动输入买入的代码、日期和价格。
- **盈亏追踪：** 自动获取最新价格并实时计算浮动盈亏（Unrealized P/L）。
- **历史记录：** 记录已平仓的每一笔交易，并统计整体**胜率**。

### 4.2 智能退出分析 (Aim 图标)
这是本系统的特色功能，通过“退出建议弹窗”，您可以检查当前是否触发了：
- **ATR 动态止损：** 基于波动率的止损线。
- **目标获利：** 达到预期的利润目标。
- **时间止损：** 达到预定持仓天数且盈利不达标。

## 5. 数据中心 (Data Center)
**位置：** 导航栏 -> 数据中心

本系统运行的基础，负责管理本地历史数据库。

- **数据同步：** 
  - **模式：** 支持全市场扫描或指定个股同步。
  - **数据源：** 可选择 YFinance (全球兼容) 或 AkShare (A股深度支持)。
  - **增量更新：** 勾选后仅下载自上次更新起的新数据，效率更高。
- **缺失告警：** 红色高亮展示由于停牌或同步失败导致数据不再更新的标的。

---

## 6. 配置中心 (Config Center)
**位置：** 导航栏 -> 配置中心

用于个性化您的验证逻辑。

- **ATR 策略参数：** 调整止损、止盈的乘数。
- **持有周期：** 设定最大持股天数。
- **浏览器持久化：** 这里的配置**仅保存在您的本地浏览器**中，确保了您的个人策略偏好不会被干扰到后端模型。
