# 股票信息扩展系统

## 概述

使用yfinance API扩展股票基本信息数据库，为量化因子构建提供更丰富的基本面数据。

## 数据库结构

### stock_info_extended 表

扩展表包含以下类别的信息：

#### 1. 基本信息
- `sector`: 行业板块
- `industry`: 细分行业
- `market_cap`: 市值

#### 2. 估值指标
- `pe_ratio`: 市盈率（TTM）
- `pb_ratio`: 市净率
- `forward_pe`: 预期市盈率
- `peg_ratio`: PEG比率
- `price_to_sales`: 市销率
- `price_to_book`: 市净率
- `enterprise_to_revenue`: 企业价值/营收
- `enterprise_to_ebitda`: 企业价值/EBITDA

#### 3. 财务指标
- `profit_margins`: 利润率
- `gross_margins`: 毛利率
- `operating_margins`: 营业利润率
- `ebitda_margins`: EBITDA利润率
- `return_on_assets`: ROA（资产回报率）
- `return_on_equity`: ROE（净资产收益率）

#### 4. 资产负债
- `total_cash`: 总现金
- `total_debt`: 总债务
- `debt_to_equity`: 资产负债率
- `current_ratio`: 流动比率
- `quick_ratio`: 速动比率
- `book_value`: 账面价值

#### 5. 现金流
- `free_cashflow`: 自由现金流
- `operating_cashflow`: 经营现金流

#### 6. 增长指标
- `revenue_growth`: 营收增长率
- `earnings_growth`: 盈利增长率
- `earnings_quarterly_growth`: 季度盈利增长率

#### 7. 股息指标
- `dividend_rate`: 股息率
- `dividend_yield`: 股息收益率
- `payout_ratio`: 派息比率
- `five_year_avg_dividend_yield`: 5年平均股息收益率

#### 8. 风险指标
- `beta`: 贝塔系数

#### 9. 股本信息
- `shares_outstanding`: 流通股本
- `float_shares`: 自由流通股
- `held_percent_insiders`: 内部人持股比例
- `held_percent_institutions`: 机构持股比例

#### 10. 价格信息
- `current_price`: 当前价格
- `fifty_two_week_high`: 52周最高价
- `fifty_two_week_low`: 52周最低价
- `fifty_day_average`: 50日均价
- `two_hundred_day_average`: 200日均价

#### 11. 成交量
- `average_volume`: 平均成交量
- `average_volume_10days`: 10日平均成交量

## 使用方法

### 1. 更新股票信息

```python
from scripts.update_stock_info import StockInfoUpdater

# 创建更新器
updater = StockInfoUpdater()

# 测试模式：更新前5只股票
updater.update_all_stocks(limit=5, delay=2)

# 正式模式：更新所有股票（建议在非交易时间运行）
updater.update_all_stocks(delay=1)
```

### 2. 查询股票信息

```python
from scripts.query_stock_info import StockInfoQuery

# 创建查询器
query = StockInfoQuery()
query.connect_db()

# 获取所有股票
all_stocks = query.get_all_stocks()

# 根据代码查询
stock = query.get_stock_by_code('000001')

# 筛选低估值股票
low_valuation = query.filter_by_valuation(pe_max=20, pb_max=2)

# 筛选高增长股票
high_growth = query.filter_by_growth(
    revenue_growth_min=0.2,  # 营收增长>20%
    earnings_growth_min=0.2   # 盈利增长>20%
)

# 筛选高盈利能力股票
high_profit = query.filter_by_profitability(
    roe_min=0.15,  # ROE>15%
    roa_min=0.05   # ROA>5%
)

# 导出到CSV
query.export_to_csv('stock_info.csv')

query.close_db()
```

### 3. 命令行使用

```bash
# 更新股票信息
python scripts/update_stock_info.py

# 查询和统计
python scripts/query_stock_info.py
```

## 量化因子应用

这些扩展信息可用于构建以下量化因子：

### 1. 价值因子
- PE、PB、PS等估值指标
- 企业价值倍数（EV/EBITDA、EV/Revenue）

### 2. 质量因子
- ROE、ROA等盈利能力指标
- 毛利率、营业利润率等利润率指标
- 资产负债率、流动比率等财务健康指标

### 3. 成长因子
- 营收增长率、盈利增长率
- 季度增长率

### 4. 动量因子
- 52周高低点相对位置
- 50日/200日均线相对位置

### 5. 风险因子
- Beta系数
- 波动率相关指标

### 6. 股息因子
- 股息率、派息比率
- 历史股息稳定性

## 注意事项

1. **数据源限制**：yfinance对A股数据的支持可能不如美股完整，部分字段可能为空
2. **请求频率**：建议设置适当的延迟（1-2秒），避免请求过快被限制
3. **更新时机**：建议在非交易时间（如晚上或周末）进行批量更新
4. **数据质量**：使用前建议检查数据完整性和准确性
5. **代码转换**：脚本自动将A股代码转换为yfinance格式（6开头加.SS，其他加.SZ）

## 数据更新策略

建议的更新频率：

- **日度更新**：价格信息、成交量
- **周度更新**：估值指标（PE、PB等）
- **月度更新**：财务指标、增长指标
- **季度更新**：完整的基本面数据

## 故障排查

### 问题1：无法获取某些股票数据
- 原因：yfinance对部分A股支持不完整
- 解决：检查股票代码格式，或使用备用数据源

### 问题2：请求被限制
- 原因：请求频率过高
- 解决：增加delay参数值

### 问题3：数据字段为空
- 原因：该股票该字段数据不可用
- 解决：使用时需要处理None值，或使用默认值填充
