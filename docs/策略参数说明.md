# 策略参数配置说明

## 概述

两个策略文件（`wyckoff_strategy.py` 和 `smc_liquidity_strategy.py`）的所有硬编码参数已经抽象到文件开头，方便回测时调整。

## Wyckoff Spring 策略参数

### 位置
文件：`wyckoff_strategy.py`
参数位置：文件开头（第10-60行）

### 主要参数分类

#### 1. 数据周期参数
- `MIN_DATA_DAYS = 60` - 最少需要的数据天数

#### 2. 均线参数
- `MA_SHORT_PERIOD = 5` - 短期均线周期
- `MA_MID_PERIOD = 20` - 中期均线周期
- `MA_LONG_PERIOD = 60` - 长期均线周期
- `MA_SLOPE_LOOKBACK = 6` - 均线斜率计算回看天数
- `MA_PRICE_TOLERANCE = 0.02` - 价格接近均线的容差（2%）
- `MA_SPREAD_MAX = 0.15` - 均线间距最大值（15%）
- `MA_MIN_STRENGTH = 50` - 均线向上最低强度要求

#### 3. 横盘区域参数
- `CONSOLIDATION_DAYS = 20` - 横盘区域检测天数
- `CONSOLIDATION_RANGE = 0.15` - 横盘区域价格波动范围（15%）

#### 4. 成交量参数
- `VOLUME_SHRINK_RATIO = 0.8` - 成交量萎缩比例
- `VOLUME_SURGE_RATIO = 1.5` - Spring形态成交量放大比例

#### 5. Spring形态检测参数
- `SPRING_LOOKBACK_DAYS = 10` - Spring形态检测回看天数
- `SPRING_SUPPORT_LOOKBACK = 30` - 支撑位计算回看天数
- `SPRING_BREAK_THRESHOLD = 0.99` - 跌破支撑阈值（1%）

#### 6. RSI参数
- `RSI_OVERSOLD_MIN = 30` - RSI超卖下限
- `RSI_OVERSOLD_MAX = 50` - RSI超卖上限

#### 7. MACD背离检测参数
- `MACD_DIVERGENCE_LOOKBACK = 20` - MACD背离检测回看天数
- `MACD_DIVERGENCE_SHORT = 10` - MACD背离短期对比天数

#### 8. 置信度阈值
- `CONFIDENCE_BUY = 60` - 买入信号置信度阈值
- `CONFIDENCE_WATCH = 40` - 观察信号置信度阈值

#### 9. 止损和目标参数
- `STOP_LOSS_BUFFER = 0.015` - 止损缓冲（1.5%）
- `TARGET_PROFIT = 0.12` - 目标收益（12%）

#### 10. 置信度权重
- `CONFIDENCE_CONSOLIDATION = 20` - 横盘区域确认权重
- `CONFIDENCE_VOLUME_SHRINK = 15` - 成交量萎缩权重
- `CONFIDENCE_SPRING = 25` - Spring形态权重
- `CONFIDENCE_RSI = 15` - RSI回升权重
- `CONFIDENCE_MACD = 25` - MACD背离权重

## SMC 流动性猎取策略参数

### 位置
文件：`smc_liquidity_strategy.py`
参数位置：文件开头（第10-120行）

### 主要参数分类

#### 1. 数据周期参数
- `LOOKBACK_PERIOD = 90` - 回看周期（天）
- `TREND_ANALYZER_LOOKBACK = 120` - 趋势线分析回看周期（天）

#### 2. 流动性猎取参数
- `LIQUIDITY_SWEEP_THRESHOLD = 0.02` - 流动性扫荡阈值（2%）
- `SPRING_RECOVERY_BARS = 5` - Spring恢复K线数量

#### 3. 订单块参数
- `ORDER_BLOCK_STRENGTH = 1.5` - 订单块成交量强度倍数
- `ORDER_BLOCK_BREAKTHROUGH = 0.02` - 突破幅度（2%）
- `ORDER_BLOCK_DISTANCE = 0.03` - 订单块回踩距离（3%）

#### 4. FVG参数
- `FVG_MIN_GAP_RATIO = 0.001` - FVG最小缺口比例（0.1%）
- `FVG_MAX_FILLED = 0.5` - FVG最大回填比例（50%）
- `FVG_DISTANCE = 0.05` - FVG距离阈值（5%）

#### 5. 均线参数
- `TREND_MA_LONG_PERIOD = 90` - 长期均线周期
- `TREND_MA_MID_PERIOD = 30` - 中期均线周期
- `TREND_MA_SHORT_PERIOD = 5` - 短期均线周期

#### 6. 趋势判断参数
- `TREND_MA_LONG_SLOPE_LOOKBACK = 20` - 长期均线斜率回看天数
- `TREND_MA_MID_SLOPE_LOOKBACK = 10` - 中期均线斜率回看天数
- `TREND_MA_SHORT_SLOPE_LOOKBACK = 3` - 短期均线斜率回看天数

#### 7. 成交量参数
- `VOLUME_SURGE_RATIO = 1.2` - 成交量放大倍数

#### 8. ATR参数

- `ATR_STOP_MULTIPLIER = 1.5` - ATR止损倍数
- `ATR_TARGET_MULTIPLIER = 3` - ATR目标倍数

#### 9. 入场时机参数

- `ENTRY_MAX_VOLUME_RATIO = 3.0` - 最大成交量倍数
- `ENTRY_MAX_DISTANCE_FROM_SUPPORT = 0.08` - 距离支撑最大距离（8%）

#### 10. 止损和目标参数
- `MIN_RISK_REWARD_RATIO = 1.5` - 最小风险收益比
- `TARGET_RISK_MULTIPLIER = 2.5` - 目标风险倍数
- `TARGET_PROFIT_PCT = 0.08` - 目标收益百分比（8%）
- `STOP_LOSS_MIN_PCT = 0.03` - 最小止损百分比（3%）

- `STRONG_TREND_STOP_MAX = 0.96` - 强趋势最大止损（4%）
- `WEAK_TREND_STOP_MIN = 0.96` - 弱趋势最小止损（4%）
- `WEAK_TREND_STOP_MAX = 0.98` - 弱趋势最大止损（2%）

#### 12. 置信度阈值

- `CONFIDENCE_STRONG_BUY = 85` - 强买入信号置信度

#### 13. 趋势线参数

#### 14. 趋势强度评分阈值

- `TREND_STRENGTH_MODERATE = 50` - 中等趋势阈值
- `TREND_STRENGTH_WEAK = 40` - 弱趋势阈值

- `BEARISH_MIN_CONFIDENCE = 50` - 看空信号最低置信度

## 如何调整参数进行回测

### 方法1：直接修改文件开头的常量

1. 打开策略文件（`wyckoff_strategy.py` 或 `smc_liquidity_strategy.py`）
2. 找到文件开头的参数配置区域（有明显的注释标记）
3. 修改你想要调整的参数值
4. 保存文件
5. 运行回测

### 方法2：创建参数配置文件（推荐用于批量测试）

如果需要测试多组参数，可以考虑：

1. 创建一个参数配置文件（如 `strategy_params.json`）
2. 在策略初始化时读取配置文件
3. 批量运行不同参数组合的回测

### 参数调整建议

#### 提高信号质量（减少假信号）
- 提高置信度阈值：`CONFIDENCE_BUY`, `MIN_CONFIDENCE`
- 提高趋势强度要求：`MIN_TREND_STRENGTH`, `MA_MIN_STRENGTH`
- 提高入场质量要求：`MIN_ENTRY_QUALITY`

#### 增加信号数量（可能增加假信号）
- 降低置信度阈值
- 放宽均线容差：`MA_PRICE_TOLERANCE`, `TREND_PRICE_VS_LONG_TOLERANCE`
- 放宽入场时机限制：`ENTRY_MAX_DAILY_RETURN`, `ENTRY_MAX_DISTANCE_FROM_SUPPORT`

#### 调整风险收益比
- 修改止损参数：`STOP_LOSS_BUFFER`, `ATR_STOP_MULTIPLIER`
- 修改目标参数：`TARGET_PROFIT`, `TARGET_RISK_MULTIPLIER`
- 修改最小风险收益比：`MIN_RISK_REWARD_RATIO`

#### 优化趋势识别
- 调整均线周期：`MA_SHORT_PERIOD`, `MA_MID_PERIOD`, `MA_LONG_PERIOD`
- 调整均线斜率回看天数：`MA_SLOPE_LOOKBACK`, `TREND_MA_*_SLOPE_LOOKBACK`
- 调整趋势强度阈值：`TREND_STRENGTH_*`

## 注意事项

1. **参数相互影响**：某些参数之间存在关联，修改一个可能需要相应调整其他参数
2. **过度优化风险**：避免过度拟合历史数据，保持参数的合理性
3. **回测验证**：每次修改参数后都应该进行充分的回测验证
4. **市场适应性**：不同市场环境可能需要不同的参数设置
5. **保留原始值**：建议在修改前记录原始参数值，便于对比和回退
