# 新回测系统说明

## 概述

在 `core/backtest` 中实现了全新的模块化回测系统，相比旧系统（`quantitative_retest`）具有更好的可扩展性和代码组织。

## 系统架构

### 核心模块

```
core/backtest/
├── engine.py              # 回测引擎（协调器）
├── strategy.py            # 策略基类和信号定义
├── portfolio.py           # 投资组合管理
├── data_handler.py        # 数据处理器
├── performance.py         # 性能分析器
├── strategies/            # 策略实现
│   ├── __init__.py
│   ├── ml_factor_strategy.py    # ML因子策略
│   └── example_strategy.py      # 示例策略
└── README.md             # 详细文档
```

### 设计原则

1. **单一职责** - 每个模块只负责一个功能
2. **开闭原则** - 对扩展开放，对修改关闭
3. **依赖倒置** - 依赖抽象而非具体实现
4. **接口隔离** - 清晰的接口定义

## 主要特点

### 1. 模块化设计

- **DataHandler**: 负责数据加载、缓存和查询
- **Portfolio**: 管理持仓、现金和交易
- **Strategy**: 策略逻辑独立
- **PerformanceAnalyzer**: 性能分析和报告

### 2. 策略可扩展

只需继承 `BaseStrategy` 并实现 `generate_signals` 方法：

```python
class MyStrategy(BaseStrategy):
    def generate_signals(self, current_date, market_data, portfolio_state):
        # 策略逻辑
        return [signal1, signal2, ...]
```

### 3. 清晰的数据流

```
DataHandler -> Strategy -> Portfolio -> PerformanceAnalyzer
     ↓           ↓            ↓              ↓
   加载数据    生成信号    执行交易      计算指标
```

### 4. 高性能

- 并行数据加载
- 日期索引优化
- 数据缓存机制

## 使用示例

### 基本使用

```python
from core.backtest import BacktestEngine, DataHandler
from core.backtest.strategies import MLFactorBacktestStrategy

# 1. 创建数据处理器
data_handler = DataHandler('data/stock_data.db')

# 2. 创建策略
strategy = MLFactorBacktestStrategy(
    model_path='models/xgboost_factor_model.pkl',
    min_confidence=60.0
)

# 3. 创建回测引擎
engine = BacktestEngine(
    strategy=strategy,
    data_handler=data_handler,
    initial_capital=1.0,
    commission_rate=0.001
)

# 4. 运行回测
results = engine.run('2023-01-01', '2024-12-31')

# 5. 查看结果
print(results['metrics'])
```

### 自定义策略

```python
from core.backtest.strategy import BaseStrategy, StrategySignal

class MyStrategy(BaseStrategy):
    def __init__(self):
        super().__init__(name="我的策略")
    
    def generate_signals(self, current_date, market_data, portfolio_state):
        signals = []
        
        for stock_code, data in market_data.items():
            # 策略逻辑
            if self._should_buy(data):
                signal = StrategySignal(
                    stock_code=stock_code,
                    signal_type='buy',
                    timestamp=current_date,
                    price=data['close'].iloc[-1],
                    confidence=80.0
                )
                signals.append(signal)
        
        return signals
```

## 内置策略

### 1. ML因子策略 (MLFactorBacktestStrategy)

基于机器学习因子模型的选股策略。

**特点**:
- 使用训练好的ML模型预测
- 支持置信度过滤
- 自动计算止损止盈

**使用**:
```python
strategy = MLFactorBacktestStrategy(
    model_path='models/xgboost_factor_model.pkl',
    min_confidence=60.0
)
```

### 2. 简单均线策略 (SimpleMovingAverageStrategy)

基于均线金叉死叉的经典策略。

**特点**:
- 短期均线上穿长期均线买入
- 可配置均线周期

**使用**:
```python
from core.backtest.strategies.example_strategy import SimpleMovingAverageStrategy

strategy = SimpleMovingAverageStrategy(
    short_window=5,
    long_window=20
)
```

### 3. 动量策略 (MomentumStrategy)

基于价格动量的策略。

**特点**:
- 选择过去N天涨幅最大的股票
- 可配置动量阈值

**使用**:
```python
from core.backtest.strategies.example_strategy import MomentumStrategy

strategy = MomentumStrategy(
    lookback_days=20,
    momentum_threshold=0.10
)
```

### 4. 突破策略 (BreakoutStrategy)

基于价格突破的策略。

**特点**:
- 价格突破N日最高价
- 需要成交量配合

**使用**:
```python
from core.backtest.strategies.example_strategy import BreakoutStrategy

strategy = BreakoutStrategy(
    lookback_days=20,
    volume_threshold=1.5
)
```

## 性能指标

系统自动计算以下指标：

### 交易统计
- 总交易次数
- 盈利/亏损交易数
- 胜率
- 平均持仓天数

### 收益指标
- 总收益/收益率
- 平均收益率
- 平均盈利/亏损

### 风险指标
- 盈亏比
- 最大回撤
- 夏普比率

### 其他
- 退出原因统计
- 最佳/最差交易

## 退出机制

系统内置以下退出机制：

1. **止损** - 价格触及止损位
2. **止盈** - 价格触及止盈位
3. **时间止损** - 持仓5天未盈利且亏损超过1%
4. **趋势破位** - 跌破支撑趋势线

## 扩展开发

### 添加新策略

1. 在 `core/backtest/strategies/` 创建新文件
2. 继承 `BaseStrategy`
3. 实现 `generate_signals` 方法
4. 在 `__init__.py` 中导出

示例：

```python
# my_strategy.py
from core.backtest.strategy import BaseStrategy, StrategySignal

class MyStrategy(BaseStrategy):
    def __init__(self):
        super().__init__(name="我的策略")
    
    def initialize(self, **kwargs):
        super().initialize(**kwargs)
        # 初始化逻辑
    
    def generate_signals(self, current_date, market_data, portfolio_state):
        # 策略逻辑
        return signals
    
    def cleanup(self):
        # 清理逻辑
        pass
```

## 常见问题

### Q: 如何使用多个持仓？

A: 设置 `max_positions` 参数：
```python
engine = BacktestEngine(..., max_positions=5)
```

### Q: 如何只回测特定股票？

A: 在 `run` 方法中传入股票列表：
```python
results = engine.run(
    start_date='2023-01-01',
    end_date='2024-12-31',
    stock_codes=['000001', '000002']
)
```

### Q: 如何保存回测结果？

A: 使用 `PerformanceAnalyzer`：
```python
from core.backtest.performance import PerformanceAnalyzer

PerformanceAnalyzer.save_trades_to_csv(results['trades'], 'trades.csv')
PerformanceAnalyzer.save_equity_curve(results['equity_curve'], 'equity.csv')
```

### Q: 如何调整手续费？

A: 设置 `commission_rate` 参数：
```python
engine = BacktestEngine(..., commission_rate=0.002)  # 0.2%
```
