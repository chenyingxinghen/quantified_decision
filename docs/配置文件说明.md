# 配置文件说明

## 配置文件结构

项目包含三个主要配置文件：

```
config/
├── config.py              # 基础配置（数据库路径、数据源等）
├── strategy_config.py     # 策略配置（回测、趋势线分析等）
└── factor_config.py       # 因子模型配置（模型参数、训练参数、因子参数）
```

---

## 1. config.py - 基础配置

### 用途
- 数据库路径
- 数据源配置
- 选股条件

### 主要配置项

```python
# 数据库配置
DATABASE_PATH = "data/stock_data.db"
YEARS = 10  # 历史数据年数

# 选股条件
SELECTION_CRITERIA = {
    'min_market_cap': 100,    # 最小市值（亿）
    'max_pe': 80,             # 最大市盈率
    'min_price': 1,           # 最小股价
    'max_price': 100,         # 最大股价
}
```

### 使用方法

```python
from config import DATABASE_PATH, SELECTION_CRITERIA

# 连接数据库
conn = sqlite3.connect(DATABASE_PATH)

# 使用选股条件
min_cap = SELECTION_CRITERIA['min_market_cap']
```

## 2. strategy_config.py - 策略配置

### 用途
- 回测系统参数
- 趋势线分析参数
- 风险控制参数

### 主要配置项

```python
# 风险控制
ATR_STOP_MULTIPLIER = 1       # ATR止损倍数
ATR_TARGET_MULTIPLIER = 4     # ATR目标倍数

# 回测参数
ENABLE_TAKE_PROFIT_EXIT = False
ENABLE_SUPPORT_BREAK_EXIT = True
TIME_STOP_DAYS = 5

# 趋势线参数
TREND_LINE_LONG_PERIOD = 90
TREND_LINE_SHORT_PERIOD = 18
```

### 使用方法

```python
from config.strategy_config import (
    ATR_STOP_MULTIPLIER,
    TREND_LINE_LONG_PERIOD
)

# 计算止损价
stop_loss = entry_price - atr * ATR_STOP_MULTIPLIER
```

## 3. factor_config.py - 因子模型配置

### 用途
- 模型超参数
- 训练参数
- 因子计算参数
- 优化参数

### 配置类结构

```python
FactorModelConfig
├── ModelConfig          # 模型超参数
├── TrainingConfig       # 训练参数
├── FactorConfig         # 因子参数
└── OptimizationConfig   # 优化参数
```

### 3.1 ModelConfig - 模型超参数

```python
from config import ModelConfig

# 获取XGBoost参数
xgb_params = ModelConfig.get_model_params('xgboost')
# 返回:
# {
#     'n_estimators': 200,
#     'max_depth': 6,
#     'learning_rate': 0.05,
#     ...
# }

# 默认模型列表
models = ModelConfig.DEFAULT_MODELS
# ['xgboost', 'lightgbm', 'random_forest']
```

### 3.2 TrainingConfig - 训练参数

```python
from config import TrainingConfig

# 数据集划分
train_ratio = TrainingConfig.TRAIN_TEST_SPLIT  # 0.8
val_ratio = TrainingConfig.VALIDATION_SPLIT    # 0.2

# 标签定义
future_days = TrainingConfig.FUTURE_DAYS       # 5
threshold = TrainingConfig.RETURN_THRESHOLD    # 0.02

# 缓存设置
cache_dir = TrainingConfig.CACHE_DIR           # 'data/factors_cache'
```

### 3.3 FactorConfig - 因子参数

```python
from config import FactorConfig

# 动量因子
rsi_period = FactorConfig.RSI_PERIOD           # 6
roc_period = FactorConfig.ROC_PERIOD           # 12

# 趋势因子
macd_fast = FactorConfig.MACD_FAST             # 12
macd_slow = FactorConfig.MACD_SLOW             # 26
adx_period = FactorConfig.ADX_PERIOD           # 14

# 波动率因子
atr_period = FactorConfig.ATR_PERIOD           # 14
bb_period = FactorConfig.BB_PERIOD             # 20

# 成交量因子
volume_ma = FactorConfig.VOLUME_MA_PERIOD      # 20
mfi_period = FactorConfig.MFI_PERIOD           # 14
```

### 3.4 OptimizationConfig - 优化参数

```python
from config import OptimizationConfig

# 特征选择
method = OptimizationConfig.FEATURE_SELECTION_METHOD  # 'hybrid'
n_features = OptimizationConfig.N_FEATURES_TO_SELECT  # 30

# 集成学习
```

## 修改配置

### 方法1: 直接修改配置文件

编辑 `config/factor_config.py`:

```python
class FactorConfig:
    RSI_PERIOD = 14  # 从6改为14
    MACD_FAST = 8    # 从12改为8
```

### 方法2: 运行时覆盖

```python
from config import FactorConfig

# 临时修改（不影响配置文件）
FactorConfig.RSI_PERIOD = 14

# 或者创建自定义参数
custom_params = ModelConfig.get_model_params('xgboost')
custom_params['n_estimators'] = 300
custom_params['max_depth'] = 8
```

## 常见问题

### Q1: 为什么有些参数在多个文件中？

**A:** 不同文件服务不同目的：
- `config.py`: 全局基础配置
- `strategy_config.py`: 回测和策略相关
- `factor_config.py`: 机器学习模型相关

### Q2: 如何知道应该修改哪个配置文件？

**A:** 根据功能选择：
- 修改数据库路径 → `config.py`
- 修改回测参数 → `strategy_config.py`
- 修改模型参数 → `factor_config.py`

### Q3: 配置修改后需要重启吗？

**A:** 是的，Python会在导入时读取配置，修改后需要重新运行程序。

### Q4: 如何在不同环境使用不同配置？

**A:** 可以使用环境变量：

```python
import os

DATABASE_PATH = os.getenv(
    'DB_PATH',
    'data/stock_data.db'  # 默认值
)
```

## 最佳实践

1. **不要硬编码参数** - 始终从配置文件读取
2. **使用类访问** - 通过 `FactorConfig.RSI_PERIOD` 而不是直接导入
3. **文档化修改** - 修改配置时添加注释说明原因
4. **版本控制** - 重要的配置修改应该提交到git
5. **测试配置** - 修改后运行测试确保没有破坏现有功能

## 配置优先级

当同一参数在多处定义时，优先级为：

1. 运行时传入的参数（最高）
2. 环境变量
3. 配置文件
4. 默认值（最低）

示例：

```python
def train_model(n_estimators=None):
    # 优先级: 参数 > 配置 > 默认值
    n_est = n_estimators or ModelConfig.XGBOOST_PARAMS['n_estimators'] or 100
    ...
```
