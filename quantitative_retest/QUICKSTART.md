# 快速开始指南

## 3分钟上手回测系统

### 步骤 1: 检查环境 ✓

```bash
# 确保在项目根目录
cd quantified_decision

# 检查数据库是否存在
dir stock_data.db
```

如果数据库不存在，先运行：
```bash
python main.py
# 选择"初始化数据库"
```

### 步骤 2: 进入回测目录

```bash
cd quantitative_retest
```

### 步骤 3: 测试导入（可选）

```bash
python test_import.py
```

应该看到：
```
✓ backtest_engine 导入成功
✓ performance_analyzer 导入成功
✓ 所有模块导入成功！
```

### 步骤 4: 运行回测 🚀

```bash
python run_backtest.py
```

### 步骤 5: 选择策略

```
请选择回测策略:
1. SMC流动性猎取策略 (liquidity_grab)
2. 威科夫Spring反转策略 (wyckoff_spring)

请选择 (1-2): 1
```

输入 `1` 或 `2`

### 步骤 6: 设置时间范围

```
默认回测时间范围: 2023-01-20 至 2026-01-20
是否自定义时间范围？(y/n): n
```

输入 `n` 使用默认范围，或 `y` 自定义

### 步骤 7: 等待完成 ⏳

系统会显示进度：
```
正在加载历史数据...
成功加载 2054 只股票的历史数据
交易日数量: 725

开始回测...
[2023-01-21] 买入 600519: 1850.00 ...
进度: 50/725 交易日 | 完成交易: 5 笔 | 当前资金: 1.1234
...
回测完成！
```

### 步骤 8: 查看结果 📊

控制台会显示：
```
================================================================================
回测摘要报告
================================================================================
总收益率: 45.32%
胜率: 62.50%
总交易次数: 48
...
```

生成的文件：
- `backtest_trades_*.csv` - 交易记录
- `backtest_equity_curve_*.png` - 收益率曲线图

## 结果解读

### 总收益率
- **正数**: 策略盈利
- **负数**: 策略亏损
- **示例**: 45.32% = 初始1.0变成1.4532

### 胜率
- **高于50%**: 盈利交易多于亏损
- **低于50%**: 亏损交易多于盈利
- **示例**: 62.50% = 10笔交易中6笔盈利

### 交易次数
- **太少**: 可能策略太严格
- **太多**: 可能频繁交易
- **合理**: 20-100笔/年



- 检查数据库是否有数据
- 尝试扩大时间范围
- 降低策略置信度阈值

```bash
# 确保在正确目录
cd quantitative_retest
python test_import.py
```
