# A股量化选股系统

基于K线结构和价格行为分析的A股量化选股程序，集成全球顶级交易策略，使用免费数据源，支持实时数据更新和多种高级选股策略。

## 功能特点

### 基础功能
- **免费数据源**: 使用akshare获取A股实时和历史数据
- **技术指标**: 支持MA、RSI、MACD、布林带、KDJ等常用技术指标
- **多种策略**: 内置黄金交叉、超卖反弹、突破等选股策略
- **自定义筛选**: 支持用户自定义筛选条件
- **实时更新**: 交易时间内定时更新数据
- **本地存储**: 使用SQLite本地存储数据，提高查询效率

### 高级功能 (基于全球顶级交易策略)
- **价格行为分析**: 基于K线结构的HH-HL-LH-LL四元素趋势识别
- **威科夫理论**: Spring形态识别和积累阶段分析
- **多周期共振**: 1:4:16黄金比例模型的多时间框架分析
- **高概率形态**: 78%成功率的三只乌鸦、锤头线等经典形态识别
- **动态风险管理**: ATR动态止损和科学仓位管理
- **综合策略评分**: 多策略信号融合和置信度评估

## 核心策略详解

### 1. 威科夫积累策略 (Wyckoff Accumulation)
```
识别特征:
- 底部横盘区域确认 (波动<15%)
- Spring形态 - 跌破支撑后快速回升
- 成交量萎缩后放量确认
- RSI从超卖区域回升 (30-50)
- MACD底背离确认

成功率: 75-80%
风险收益比: 1:3
```

### 2. 黄金交叉动量策略 (Golden Cross Momentum)
```
核心条件:
- EMA20上穿SMA50黄金交叉
- 价格位于双均线上方
- 市场结构确认上升趋势 (HH-HL模式)
- MACD金叉确认
- 成交量放大 (>1.2倍平均量)

成功率: 70-75%
风险收益比: 1:2.5
```

### 3. 均值回归超卖策略 (Mean Reversion Oversold)
```
触发条件:
- RSI < 30 (超卖)
- KDJ < 20 (超卖)
- 价格接近布林带下轨
- 锤头线等反转形态确认
- 价格偏离20日均线 > 10%

成功率: 65-70%
风险收益比: 1:2
```

### 4. 突破动量策略 (Breakout Momentum)
```
突破确认:
- 价格突破关键阻力位 (>0.5%)
- 成交量大幅放大 (>2倍平均量)
- RSI处于强势区间 (50-80)
- MACD柱状图向上
- 布林带上轨突破

成功率: 68-72%
风险收益比: 1:2
```

### 5. 多周期共振策略 (Multi-Timeframe Confluence)
```
共振要求:
- 长期趋势: 上升趋势确认 (权重40%)
- 中期结构: 支持或盘整 (权重30%)
- 短期信号: 看涨信号 (权重20%)
- 关键位置: 重要支撑/阻力附近 (权重10%)

成功率: 75-80%
风险收益比: 1:3
```

## 技术指标详解

### K线结构分析
- **HH (Higher High)**: 更高高点 - 买方力量增强
- **HL (Higher Low)**: 更高低点 - 支撑位上移
- **LH (Lower High)**: 更低高点 - 阻力位下移  
- **LL (Lower Low)**: 更低低点 - 卖方力量增强

### 高概率蜡烛图形态
| 形态名称 | 成功率 | 类型 | 识别特征 |
|---------|--------|------|----------|
| 三只乌鸦 | 78% | 顶部反转 | 三根连续长阴线，实体逐渐扩大 |
| 看涨吞没 | 78% | 底部反转 | 大阳线完全包含前一阴线 |
| 锤头线 | 75% | 底部反转 | 下影线≥2倍实体，上影线很小 |
| 射击之星 | 70% | 顶部反转 | 上影线≥2倍实体，下影线很小 |
| 晨星/暮星 | 68% | 反转确认 | 三K线组合，中间小实体 |

### 六根K线漂移分析
基于连续6根K线的趋势一致性评分：
- **强上升趋势**: 评分 ≥ 4，一致性 > 0.8
- **强下降趋势**: 评分 ≤ -4，一致性 > 0.8  
- **盘整状态**: 评分 -1 到 1，一致性 < 0.4

## 常见问题

### Q: 首次运行需要多长时间？
A: 初始化上证+深圳板块约5000只股票，需要30-60分钟（取决于网络速度）。建议首次运行时选择增量更新。

### Q: 如何只分析特定板块？
A: 修改 `config.py` 中的 `DEFAULT_MARKETS`：
- 只分析上证：`['sh']`
- 只分析深圳：`['sz']`
- 分析北交所：`['bj']`

### Q: 筛选结果为空怎么办？
A: 
1. 检查数据是否最新（选项4查看数据库状态）
2. 调整 `config.py` 中的筛选条件（降低市值要求、放宽市盈率等）
3. 尝试其他策略或使用批量模式
4. 确认市场环境是否适合当前策略

### Q: 如何提高筛选速度？
A: 
1. 使用批量模式并限制分析数量
2. 减少候选股票的基本面条件
3. 使用增量更新而非全量更新
4. 定期清理数据库中的旧数据

### Q: 批量选股和单个策略有什么区别？
A: 
- **批量选股**: 一次运行所有策略，生成综合报告，找出多策略共同推荐的股票
- **单个策略**: 针对性分析，可以调整参数，适合深入研究特定策略

### Q: 如何解读置信度评分？
A: 
- **80%+**: 强烈买入信号，多个条件高度吻合
- **60-80%**: 买入信号，条件较好
- **40-60%**: 观察信号，可以关注但需谨慎
- **<40%**: 信号不明确，不建议操作

### Q: talib安装失败？
A: Windows用户需要下载预编译的whl文件安装，或使用conda安装

### Q: 数据获取失败？
A: 检查网络连接，或者数据源是否正常，可尝试更换数据源

### Q: 筛选结果为空？
A: 调整筛选条件，或者检查数据是否完整，市场环境是否适合当前策略

### Q: 策略表现不佳？
A: 检查市场环境是否匹配，调整策略参数，或组合多个策略使用

### Q: 如何提高成功率？
A: 结合基本面分析，关注市场情绪，严格执行风险管理规则
