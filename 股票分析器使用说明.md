# 股票分析器使用说明

## 功能介绍

`stock_analyzer.py` 是一个综合股票分析工具，整合了两个选股策略：
1. **SMC流动性猎取策略** - 基于机构订单流的量化策略
2. **Wyckoff Spring策略** - 识别底部区域的Spring形态

## 主要功能

### 1. 自动获取最新K线数据
- 自动更新数据库中的股票数据
- 显示最新K线信息（开盘、最高、最低、收盘、成交量）
- 计算当日涨跌幅

### 2. 双策略分析
- **SMC策略分析**：检测流动性猎取、订单块、FVG等机构足迹
- **Wyckoff策略分析**：识别底部横盘、Spring形态、MACD背离等

### 3. 综合建议
- 整合两个策略的分析结果
- 给出明确的操作建议（强烈建议买入/建议买入/观望）
- 提供具体的入场价、止损价、目标价

### 4. 风险提示
- 检测看空信号（跌破均线、看空K线形态、技术指标看空等）
- 提供风险警示

## 使用方法

### 单只股票分析

```bash
python stock_analyzer.py 600519
```

### 批量分析多只股票

```bash
python stock_analyzer.py 600519 000858 601318
```

## 操作建议说明

### 强烈建议买入
- 两个策略同时给出买入信号
- 综合置信度高
- 风险收益比良好

### 建议买入
- 一个策略给出买入信号，置信度较高（≥80%）
- 或两个策略之一给出信号，但置信度一般

### 可以考虑买入
- 一个策略给出买入信号，但置信度一般（<80%）

### 观望
- 两个策略均未给出明确买入信号
- 或检测到明显看空信号

## 买入卖出规则

### 买入规则
1. **入场时机**：在建议入场价附近分批买入
2. **仓位控制**：单只股票不超过总资金的10-20%
3. **分批建仓**：可以分2-3次买入，降低风险

### 卖出规则
1. **止损**：价格跌破止损位立即止损，不要犹豫
2. **止盈**：达到目标价分批止盈（如50%仓位在目标价，剩余仓位看情况）
3. **趋势破坏**：如果趋势结构被破坏（如跌破关键均线），考虑减仓或清仓
4. **看空信号**：如果检测到明显看空信号，考虑提前止盈或减仓

### 持仓管理
1. **移动止损**：价格上涨后，可以将止损位上移到成本价或更高
2. **分批止盈**：不要一次性全部卖出，可以分批止盈
3. **定期复查**：每天收盘后重新运行分析，检查信号是否仍然有效

## 注意事项

1. **数据更新**：脚本会自动更新数据库中的股票数据，首次运行可能需要一些时间
2. **网络连接**：需要网络连接以获取最新数据
3. **风险提示**：本工具仅供参考，不构成投资建议，投资有风险，入市需谨慎
4. **回测验证**：建议先用历史数据回测策略效果，再实盘操作
5. **仓位控制**：严格控制仓位，不要重仓单只股票

## 常见问题

### Q: 如何判断信号的可靠性？
A: 主要看以下几点：
- 综合置信度（越高越好，建议≥70%）
- 信号强度（两个策略同时给出信号最好）
- 风险收益比（建议≥1.5）
- 是否有看空信号（有看空信号需谨慎）

### Q: 止损位可以调整吗？
A: 可以，但建议：
- 不要把止损设得太紧（容易被震出）
- 不要把止损设得太松（风险太大）
- 建议风险控制在5%以内

### Q: 什么时候应该卖出？
A: 以下情况考虑卖出：
- 价格跌破止损位（必须卖）
- 达到目标价（分批止盈）
- 趋势结构被破坏（考虑减仓）
- 检测到明显看空信号（考虑减仓或清仓）

### Q: 可以同时买入多只股票吗？
A: 可以，但要注意：
- 总仓位不要超过80%
- 单只股票不超过20%
- 分散投资，降低风险
