# 使用说明

## 三个核心脚本

### 1️⃣ init_data.py - 数据初始化脚本
**用途**：初始化和更新股票数据库

**使用方法**：
```bash
python init_data.py
```

**功能菜单**：
- 选项1：初始化股票数据（首次使用必选）
- 选项2：增量更新数据（日常更新）
- 选项3：全量更新数据（重新下载所有数据）
- 选项4：查看数据库状态

**首次使用**：
- 选择选项1初始化数据
- 默认下载上证+深圳主板约5000只股票
- 预计耗时30-60分钟

---

### 2️⃣ integrated_screener.py - 选股脚本
**用途**：主动筛选当前买点的股票

**使用方法**：
```bash
python integrated_screener.py
```

**策略选择**：
- 选项1：SMC流动性猎取策略
  - 特点：快速反弹，高胜率
  - 适合：短期交易
  
- 选项2：威科夫Spring策略
  - 特点：底部反转，趋势跟随
  - 适合：中长期持有
  
- 选项3：综合筛选（推荐）
  - 同时运行两个策略
  - 找出多策略共振的股票

**输出结果**：
- 生成CSV文件：`screening_[策略名]_[时间戳].csv`
- 包含：股票代码、入场价、止损价、目标价、风险收益比等

---

### 3️⃣ quantitative_retest/run_backtest.py - 回测脚本
**用途**：对策略进行历史数据回测验证

**使用方法**：
```bash
python quantitative_retest/run_backtest.py
```

**回测流程**：
1. 选择要回测的策略（SMC或威科夫）
2. 设置回测时间范围（默认近5年）
3. 系统自动运行回测并生成报告

**输出结果**：
- 交易记录CSV：`backtest_trades_[策略名]_[时间戳].csv`
- 收益曲线图：`backtest_equity_curve_[策略名]_[时间戳].png`
- 性能指标：总收益率、胜率、最大回撤、夏普比率等

---

## 使用流程

### 第一次使用
```bash
# 1. 初始化数据
python init_data.py
# 选择 1 - 初始化股票数据

# 2. 运行选股
python integrated_screener.py
# 选择 3 - 综合筛选

# 3. 回测验证
python quantitative_retest/run_backtest.py
# 选择策略并查看历史表现
```

### 日常使用
```bash
# 1. 更新数据（每天开盘前）
python init_data.py
# 选择 2 - 增量更新数据

# 2. 运行选股
python integrated_screener.py
# 选择策略进行筛选
```

## 常见问题

**Q：首次运行需要多久？**
A：初始化数据约30-60分钟，选股约5-10分钟，回测约10-20分钟。

**Q：如何解读选股结果？**
A：重点关注置信度≥70%的股票，风险收益比≥2:1的机会。

**Q：回测结果准确吗？**
A：回测基于历史数据，仅供参考。实盘交易需考虑滑点、流动性等因素。

**Q：可以同时运行多个脚本吗？**
A：不建议。数据库可能被锁定，建议按顺序执行。
